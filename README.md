# CorelDRAW

## Overview

CorelDRAW is a graphic design software developed for Windows users, aimed at creating and editing vector graphics and page layouts. It supports various design tasks including illustration, photo editing, and typography management, providing tools suited for professional and personal projects.

## Key Features

**Vector illustration and editing tools**  
**Page layout and multi-page document support**  
**Advanced typography controls**  
**Image editing and enhancement options**  
**Color management and palette customization**  
**Support for multiple file formats**  
**Customizable workspace for improved workflow**  
**Precision drawing and layout alignment tools**
